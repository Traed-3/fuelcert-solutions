▸ Field Query
"What is the maximum allowable pressure loss for a fiberglass piping system test per PEI RP1200?"
Per PEI RP1200-19 Section 5.4.2: For pressurized fiberglass piping tested at 1.5× MOP, the system must hold for 30 minutes with zero pressure drop to pass. Any measurable loss is a failure and requires leak investigation before backfill.
Source: PEI/RP1200-19 · Section 5.4.2 · Tightness Testing
▸ Field Query
"WV WVDEP — what is the minimum passing score for a Class C Tester exam?"
West Virginia DEP requires a minimum score of 80% to pass any UST certification examination, including Class C (Tester). Candidates scoring below 80% must retake the full exam.
Source: WV UST Certification Program · Class C Study Guide
▸ Field Query
"Maryland — what does COMAR 26.10 require for spill bucket testing frequency?"
Under COMAR 26.10.07, spill prevention equipment must be tested at installation and at least once every 3 years thereafter, or annually if the containment sump does not have a liquid-tight design verified by a third party.
Source: COMAR 26.10.07 · MDE UST Compliance Guide
