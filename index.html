▸ FIELD QUERY
"What is the maximum allowable pressure loss for a fiberglass piping system test using PEI RP1200?"
Per PEI RP1200-19, Section 5.4.2: For pressurized fiberglass piping tested at 1.5× MOP, the system must hold for 30 minutes with zero pressure drop to pass. Any measurable loss constitutes a failure and requires leak investigation before backfill.

Source: PEI/RP1200-19 · Section 5.4.2 · Tightness Testing
▸ FIELD QUERY
"WV WVDEP Class C — what is the minimum passing score?"
The West Virginia DEP requires a minimum score of 80% to pass any UST certification examination, including Class C (Tester). Candidates scoring below 80% must retake the exam.

Source: WV UST Certification Program · Class C Study Guide
